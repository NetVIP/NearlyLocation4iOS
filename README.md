# NearlyLocation4iOS
附近的地址。通过列表显示附近的地理位置。类似于微信朋友圈发布时的选择地点。

# 地图服务商
高德地图

# 如何运行
1. 创建PrefixHeader.pch文件
2. 在其中添加 #define NLMapKey @"在高德地图申请的Key" （*请注意修改Bundle Identifier）
3. 使用pod install安装插件
